# Experiencia Inmersiva 360° - Proyecto SENNOVA

## Descripción

Proyecto desarrollado dentro del programa SENNOVA enfocado en la creación de una experiencia virtual interactiva para una institución religiosa ubicada en Candelaria, Valle del Cauca.

La solución permite a los usuarios recorrer diferentes espacios mediante fotografías 360°, accediendo a información adicional a través de elementos interactivos.

## Tecnologías utilizadas

- Marzipano
- JavaScript
- HTML
- CSS

## Funcionalidades

- Recorridos virtuales 360°
- Hotspots interactivos
- Info-hotspots informativos
- Navegación entre escenas
- Experiencia web inmersiva
